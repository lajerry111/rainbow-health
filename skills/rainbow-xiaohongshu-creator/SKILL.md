# Rainbow Xiaohongshu Creator

专门创作小红书风格的健康生活方式内容，帮助用户分享健康数据、餐食、运动、产品种草等内容。

## Capabilities

### 1. 内容类型

支持多种健康类内容创作：

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| 🍽️ 健康餐分享 | 营养餐食打卡 | 减脂餐、抗炎餐、轻食 |
| 📊 健康数据 | 体重、体脂趋势 | 里程碑分享、阶段性总结 |
| 💪 运动打卡 | 健身记录分享 | 运动日常、训练成果 |
| 🌱 产品种草 | 健康好物推荐 | 减脂食品、健康设备 |
| 🧘 生活方式 | 健康习惯养成 | 作息、冥想、养生 |

### 2. 小红书风格特点

- **标题**：情绪化、数字化、悬念式
- **正文**：emoji 丰富、分段清晰、个人化
- **标签**：精准 + 热门标签组合
- **语气**：亲切、真实、分享感

### 3. 内容优化

根据健康数据自动生成：

- 数据可视化描述（如体脂下降曲线）
- 阶段性成果总结
- 健康小贴士
- 互动引导语

## Usage

### 分享健康餐

```
@rainbow-xiaohongshu-creator 帮我写一份减脂午餐的小红书文案
食物：鸡胸肉沙拉、糙米饭、酸奶
评分：85分
```

### 分享体重里程碑

```
@rainbow-xiaohongshu-creator 分享体重突破
当前体重：65kg（已减10kg）
时间：3个月
体脂率：从30%降到22%
```

### 产品种草

```
@rainbow-xiaohongshu-creator 种草一款体脂秤
产品：彩虹生活智能体脂秤
特点：15项身体数据、APP同步、趋势分析
```
