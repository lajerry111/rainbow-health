# Rainbow Meal Photo Scorer

通过 AI 视觉分析餐食照片，从5个维度进行营养健康评分，并生成适合社交分享的健康内容。

## Capabilities

### 1. 餐食照片分析

使用 AI 视觉能力分析照片中的食物：

- 识别食物种类和分量
- 估算营养成分（热量、蛋白质、碳水、脂肪、纤维）
- 识别烹饪方式（油炸、清蒸、生食等）
- 评估食物新鲜度和色彩搭配

### 2. 五维营养评分

| 维度 | 满分 | 评分标准 |
|------|------|----------|
| 🍬 控糖 | 20分 | 低GI食物比例、添加糖含量、精制碳水比例 |
| 🥗 营养结构 | 20分 | 蛋白质充足度、优质脂肪、维生素矿物质 |
| 🔥 抗炎 | 20分 | 抗氧化物、Omega-3、彩色蔬菜比例 |
| ⚖️ 热量控制 | 20分 | 总热量合理性、份量控制 |
| 🌈 食物多样性 | 20分 | 食物种类数量、颜色多样性 |

**总分等级**
- 90-100分：💎 钻石级健康餐
- 80-89分：🥇 黄金级健康餐
- 70-79分：🥈 白银级健康餐
- 60-69分：🥉 青铜级健康餐
- <60分：⚠️ 需要改进

### 3. 个性化建议

根据用户的健康目标提供建议：

- 减脂目标：热量控制建议、饱腹感优化
- 增肌目标：蛋白质优化、营养时机
- 控糖目标：GI值优化、血糖平稳策略
- 抗炎目标：抗炎食物推荐、炎症因子控制

### 4. 小红书风格内容生成

生成适合小红书分享的内容：

- 吸引人的标题
- 营养亮点提炼
- 健康小贴士
- 适合的标签（hashtags）
- 配图建议

## Usage

### 分析餐食照片

```
[上传照片]
@rainbow-meal-photo-scorer 分析这顿饭的营养
```

### 指定健康目标

```
[上传照片]
@rainbow-meal-photo-scorer 分析这顿饭，我正在减脂
```

### 生成小红书内容

```
[上传照片]
@rainbow-meal-photo-scorer 生成小红书分享文案
```
